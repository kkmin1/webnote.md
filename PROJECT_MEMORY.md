# Scratch Project Memory

This file stores shared working preferences for projects under `C:\Users\kkmin\.gemini\antigravity\scratch`.

## Default Expectations

- Read `AGENTS.md` in the `scratch` root before substantial work in any scratch subfolder.
- Before making substantial edits, check whether the current workspace is a git repository.
- If the workspace is not a git repository, initialize git automatically before large or risky changes.
- After meaningful edits, create a git commit scoped to the current task whenever possible.
- If the git root is wider than the current project, stage only task-related files before committing.
- Keep web and desktop/Electron variants separated when their runtime requirements diverge.
- Prefer preserving prior working states when splitting or restructuring projects.
- When changing save/export behavior, verify whether both web and desktop modes are affected.
