# KS 이미지 에디터

범용 SVG 기반 이미지/다이어그램 편집기입니다. 도형, 선, 텍스트, 이미지, 표를 웹에서 직접 그리고 편집할 수 있으며, 텍스트와 표 셀 안에서 간단한 LaTeX 수식을 KaTeX로 렌더링할 수 있습니다.

---

## 실행 방법

### 로컬 서버 (권장)
```bash
# Python 3
python -m http.server 4912

# Node.js (npx)
npx serve .
```
Python 서버를 쓴다면 브라우저에서 `http://127.0.0.1:4912/index.html`로 접속합니다. `npx serve`를 쓰는 경우 터미널에 표시되는 주소로 접속합니다.

### 단일 파일로 배포
파일을 분리하지 않고 하나의 HTML로 합치면 `file://`로 바로 열 수 있습니다.
빌드 스크립트 없이 단순히 내용을 index.html에 인라인으로 넣으면 됩니다.

## 파일 구조

```
web-image-editor-object/
├── index.html        HTML 구조 (마크업만)
├── style.css         전체 스타일
├── js/
│   ├── state.js      전역 상태 & 데이터 구조 (오브젝트 스키마)
│   ├── render.js     SVG 렌더링 엔진 & 줌/캔버스 크기
│   ├── tools.js      도구 동작, 이벤트, 오른쪽 패널 동기화
│   ├── io.js         파일 저장/불러오기/내보내기
│   └── app.js        초기화, 키보드 단축키, 전역 이벤트
└── README.md
```

**로드 순서**: `state.js` → `render.js` → `tools.js` → `io.js` → `app.js`
(뒤쪽 파일이 앞쪽 파일의 함수/변수를 참조)

## 기능 목록

### 도형 도구
| 도구 | 단축키 | 설명 |
|------|--------|------|
| 선택 | `S` | 클릭 선택, 드래그 이동, 핸들로 크기 조절 |
| 텍스트 | `T` | 클릭 위치에 텍스트 입력. `$...$`, `$$...$$`는 KaTeX 수식 렌더링 |
| 원 | `C` | 중심→반지름 드래그 |
| 타원 | `E` | 대각선 드래그 |
| 사각형 | `R` | 대각선 드래그 |
| 표 | — | 클릭 시 5×5 표 삽입. 최대 10×10까지 행/열 조절 |
| 직선 | `L` | 드래그 |
| 화살표 | `A` | 끝에 화살촉 |
| 점선 | `D` | 점선 직선 |
| 점선화살표 | — | 점선 + 화살촉 |
| 자유곡선 | `F` | 마우스 경로를 따라 자유곡선 작성 |
| 2차곡선 | `Q` | 2차 베지어 곡선 |
| 3차곡선 | `B` | 3차 베지어 곡선 |

### 편집
- **선택 후 오른쪽 패널**에서 선 색상/굵기/스타일, 채우기, 투명도, 텍스트 속성, 위치/크기를 실시간 수정
- 도형 완성 후 자동 선택 모드 전환 (HWP 방식)
- 더블클릭으로 텍스트 재편집
- 표 셀 더블클릭으로 셀 내용 입력/재편집
- 텍스트 도구로 표 안을 클릭하면 독립 텍스트가 아니라 해당 셀 내용으로 입력
- 표 선택 후 오른쪽 패널에서 행/열, 셀 폭/높이 조절
- 표 내부 가로/세로 경계선을 드래그해 셀 높이/폭 조절
- 레이어 순서 변경 (앞으로/뒤로)
- Ctrl+C/V 복사/붙여넣기
- 화살표 키로 1px 미세 이동

### LaTeX 수식 입력
- 일반 텍스트와 표 셀에서 `$x^2$`, `$\frac{a}{b}$`, `$$\sum_i x_i$$` 형식을 지원
- 값은 원본 LaTeX 문자열로 저장되며, 화면에서는 KaTeX 렌더링으로 표시
- 더블클릭하면 렌더링된 수식이 아니라 원본 `$...$` 문자열을 다시 편집
- KaTeX CDN 로드에 실패하면 수식 원문 텍스트로 표시

### 파일
| 포맷 | 용도 |
|------|------|
| `.gtree` / `.json` | 편집 가능한 저장 파일. 모든 오브젝트 데이터 보존 |
| `.svg` | 벡터 이미지 내보내기 |
| `.tex` | LaTeX TikZ 코드 저장 |
| `.wtikz` | web-tikz에서 다시 편집 가능한 데이터 저장 |
| `.png` | 래스터 이미지 내보내기 (2× 해상도) |
| `.jpg` | 흰 배경 JPEG 내보내기 |
| 이미지 삽입 | PNG/JPG/GIF/WebP/SVG → 캔버스에 오브젝트로 삽입 |

### 뷰
- 줌: `+`/`−` 버튼, `Ctrl+휠`, 맞춤/1:1
- 캔버스 크기: 우하단 핸들로 드래그 조절 (기본 1600×1200)
- 사이드바 너비: 가운데 구분선 드래그로 조절

## 새 도형 타입 추가 방법

1. **`js/render.js`** — `makeEl()` 함수에 새 타입 분기 추가
2. **`js/state.js`** — `bbox()`, `hitTest()`, `moveObj()`, `copyPos()` 에 새 타입 처리 추가
3. **`js/tools.js`** — `makeNewObj()`, `updateDrawing()`, `isDegenerate()`, `applyHandle()`, `syncProps()` 에 추가
4. **`index.html`** — 왼쪽 패널 도구 그리드에 버튼 추가
5. **`js/app.js`** — `TOOL_KEYS` 에 단축키 추가 (선택)

## 기술 스택

- **Vanilla JS** (프레임워크 없음)
- **SVG** 기반 렌더링
- **CSS Flexbox** 레이아웃
- **KaTeX CDN**: `$...$`, `$$...$$` 수식 렌더링
