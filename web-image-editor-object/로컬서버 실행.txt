

  python -m http.server 4912

  그다음 접속:
  http://127.0.0.1:4912/index.html

  더 짧게 하려면 PowerShell 함수로 등록도 가능합니다:

  function wo { python -m http.server 4912 }

  이후엔 wo만 입력하면 됩니다.



› 이제 제대로 되네, 그런데 파이선 실행할 때 해당 폴더에 있어야 되는거야?


• 네. --directory를 안 쓰면 현재 위치한 폴더가 서버 루트가 됩니다.

  - 해당 폴더에서 실행:
    python -m http.server 4912
  - 다른 폴더에서 실행하려면:
    python -m http.server 4912 --directory "C:\Users\kkmin\.gemini\antigravity\scratch\directory_name"
