# web-html

이 폴더는 브라우저에서 바로 열어 사용할 수 있는 HTML 편집기 예제들을 모아 둔 scratch 작업 공간입니다.

## 성격

- `index.html`은 현재 사용 중인 통합 HTML WIZWIG 에디터입니다.
- 왼쪽에서 HTML 코드를 편집하고, 오른쪽에서 웹페이지 미리보기를 확인하는 단일 파일 웹 도구입니다.
- 표, SVG, MathJax 수식, 외부 이미지, 링크, 간단한 계산기 예제를 기본 내용으로 포함합니다.
- 파일 불러오기, 다른 이름으로 저장, 로컬 이미지 업로드 기능을 실험합니다.

## 파일 구성

- `index.html`: 최신 통합본입니다. 실제로 열어 사용할 대표 파일입니다.
- `web-html.html`: 가장 기본적인 실시간 HTML 에디터 예제입니다.
- `web-html2.html`: 파일 저장/불러오기와 MathJax 수식 예제가 추가된 버전입니다.
- `web-html3.html`: 로컬 이미지 업로드, 다른 이름으로 저장, 계산기, 태극기 SVG 예제가 포함된 버전입니다.
- `web-html4.html`: 초기 예제 계열의 보존본입니다.

## 사용 방법

브라우저에서 `index.html`을 직접 열면 됩니다.

```text
C:\Users\kkmin\.gemini\antigravity\scratch\web-html\index.html
```

## 참고 사항

- 이 폴더의 파일들은 빌드 과정 없이 동작하는 정적 HTML 파일입니다.
- `showSaveFilePicker()`를 지원하는 Chrome/Edge 계열 브라우저에서는 저장 위치 선택 창을 사용할 수 있습니다.
- 로컬 이미지 업로드는 브라우저 보안 정책 때문에 실제 로컬 경로가 아니라 현재 브라우저 세션의 `blob:` 링크로 연결됩니다. 저장한 HTML을 나중에 다시 열면 해당 이미지 링크는 유지되지 않을 수 있습니다.
- Tailwind CSS와 MathJax는 CDN을 사용하므로 인터넷 연결이 없으면 일부 스타일이나 수식 렌더링이 제한될 수 있습니다.
